About Me

This is what I do.

Hello! My name is Jennifer. Welcome to my blog website project.

This website was created as a multipart project from Udemy's 100 Days of Code Python course by Dr. Angela Yu.

This project covers the following concepts:

  • Backend development using Flask, Jinja templating, and WTForms
  • Database manipulation using SQLite and SQLAlchemy
  • RESTful API routing for adding, editing, and deleting posts
  • Security, encryption, and authentication for registering users and protecting admin only routes

Take a look around, and if you feel like registering and leaving a comment on any of the placeholder blog posts, please feel free. ;)

Note: You will need to "register" in order to leave a comment, but you can use a dummy email. Don't worry, I won't send you spam.

Thanks for visiting!
